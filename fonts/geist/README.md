# Geist webfont

This directory contains the Geist Sans variable webfont used by the homepage.

- Source: [Vercel Geist v1.7.2](https://github.com/vercel/geist-font/releases/tag/v1.7.2)
- Original file: `Geist/webfonts/Geist[wght].woff2`
- Local file: `Geist-Variable.woff2`
- Supported weight axis: `100–900`
- Homepage mapping: `400` for regular text and `700` for the title and primary links
- License: SIL Open Font License 1.1; see `OFL.txt`
- SHA-256: `a369fcf5628ea2aa4e1b9e2ec6a5b3624e365bda588e1f0f2f12b564f728fbb8`

The font is self-hosted; the live page does not depend on a font CDN.
